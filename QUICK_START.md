## 🏃 立即运行网站

### 选项1：修改 portfolio.js 使用占位符（推荐）

我已经创建了 `src/placeholderImages.js`，包含所有占位符URL。

**需要你做的：**

在 `src/portfolio.js` 文件顶部添加：
```javascript
import placeholderImages from "./placeholderImages";
```

然后将所有 `require("./assets/images/...")` 替换为对应的占位符。

**或者让我帮你自动完成这个步骤？** 

### 选项2：手动添加图片

1. 准备以下必需图片：
   - 公司Logo: polyuLogo.png, oxfordLogo.png, ietLogo.png, bornteaLogo.png
   - 项目图片: 至少4张项目封面图
   - 视频缩略图: 3张
   - 摄影封面: 3张

2. 放入 `src/assets/images/` 目录

3. 运行网站：
```bash
npm install
npm start
```
